自習用
  • URL (Uniform Resource Locator：統一資源位置指定子)
    インターネット上のWebページやファイルの位置や場所を示すもの。
    基本的に [プロトコル名] :// [ドメイン名] / [ディレクトリ名] / [ファイル名] という形式になっている。
  • SEO (Search Engine Optimization = 検索エンジン最適化)
    検索結果の上位に表示させるための手法。
  • クローラー
    インターネット上のあらゆるWebサイトの情報を取得して、検索用データベース・インデックスを作成する自動巡回プログラムのこと。WebサイトのHTMLに記載されているリンクを辿り、そのWebサイトがどのような内容であり、どのような構造になっているのか等を判断しデータベース化していく。クローラーがアクセスできるファイル・情報はすべてデータベース化され、誰もが検索できるようになってしまうので、機密文書などに関してはアクセス権を設定するなどの注意が必要である。近年では検索結果の上位に表示させるためのSEOとして、有名検索Webサイトのクローラーの動きを特に意識したWebサイト構築が目立つようになってきている。
  • オープンソース
    ソースコードを無償で一般公開すること。
  • CSS
    W3C (World Wide Web Consortium(※)) が勧告しているウェブページのレイアウトを定義する仕様である。
    ※consortium (コンソーシアム) = 共同事業体
  • ウェブ標準
    W3Cによって勧告された標準的なウェブ関連の技術を総称したものである。ウェブ標準は、どんな環境でも同じ内容の表示・表現を行うことができるようにする技術である。このウェブ標準を守る事は、アクセシビリティの向上にも繋がる。
  • アクセシビリティ
    障害のある人や高齢者を含め、誰もが使えるようにすること。
  • ブラウザ
    多少記述に誤りがあったとしても表示する機能が備わっている。
  • ユーザーインターフェース
    利用者が対象を操作するために接する部分。言い換えれば、ユーザーとウェブサイト等のインターフェース (interface = 接点) の部分。パソコンの場合、マウスやキーボード、ディスプレーといった機械的な要素、どのように操作するかという手順、画面に表示されるメニューやアイコン、ウインドウといった視覚的要素、警告音や文字の読み上げといった聴覚的要素などを指す。
  • ユーザーインターフェースデザイン (UIデザイン) 
    UI のデザインのこと。そのウェブサイトの使い易さを向上させるようなデザインを意味する。
  • ユーザーエクスペリエンス (UX) 
    ユーザー体験とも呼ばれる。あらゆる製品やサービスを通してユーザーが感じる使い易さ、感動、印象といった体験全てを指す。
  • JIS X 8341-3
    ウェブアクセシビリティについて定めた日本の規格。2004年に制定された。JISは「日本工業規格」。
  • 情報機器作業における労働衛生管理のためのガイドライン
    IT技術の進展に対応するため、厚生労働省により平成14年に策定された「VDT作業における労働衛生管理のためのガイドライン」が令和1年に改正されたもの。
  • クリック率 (CTR) 広告のクリック数を広告の表示回数 (インプレッション数) で割った値である。
    CTR = Click Through Rate
